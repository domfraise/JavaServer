Proof Of Presence Format

Top level objects:

  1. RTH type:String //The root tree hash of the tree that generated the proof
  2. Value type:String //The leaf hash value of the leaf that the proof was generated for
  3. Index type:String //The index that the leaf is stored at. Starts counting from zero, left to right
  4. Proof type:JSONObject //A JSON tree containing the proof

JSON Tree

Each node contains either Left type:JSONObject and Right type:JSONObject or Leaf type:String //the leaf value proved to be present or Hash type:String //An intermediate node value used to recreate RTH
