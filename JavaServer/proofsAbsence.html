Proof Of Presence Format

Top level objects:

  1. RTHFromDevice type:String //The root tree hash that is currently stored on the secure device
  2. DeviceSig type:String //The signature of RTHFromDevice from the device
  3. Leaves type:JSONArray //An array containing every leaf stored in the tree
  4. FileHashes type:JSONArray //An array containing all the hashes of your files
