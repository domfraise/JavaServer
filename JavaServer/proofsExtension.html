Proof Of Extension Format

Top level objects:

  1. OldRTH type:String //The root tree hash of intial, older tree
  2. NewRTH type:String //The root tree hash of newer tree to be proven to be an append only version of OldRTH tree
  3. OldProof type:JSONObject //A JSON tree containing the tree that should recreate OldRTH
  4. NewProof type:JSONObject //A JSON tree containing the tree that should recreate NewRTH

JSON Tree

Each node contains either Left type:JSONObject and Right type:JSONObject or Hash type:String //An intermediate node value used to recreate RTH
